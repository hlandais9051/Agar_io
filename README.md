
# Agar.io 
  **Agar.io** est un projet en python de fin de première année prépa CSI à l'ISEN Ouest.
  Il a été réalisé par _Matthieu Dubreil_, _Adam Launay_ en collaboration avec _Hugo Landais_ et est inspiré du jeu _Agar.io_.  
  
## Utilisation  
Pour jouer, lancez simplement le fichier ``programme.py`` dans la dernière version de python possible.
Vous retrouverez le style de l'interface dans le fichier ``interface.py``.
